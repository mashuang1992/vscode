#汉化
```
1. Ctrl+Shift+P
2.在弹出的搜索框中输入【configure language】，然后选择搜索出来的【Configure Display Language】
3.locale.json文件，可以看到locale的属性值为en
4.删除locale后面的属性直到冒号，然后重新输入冒号会自动出现代码提示
5.这里选择“zh-CN”
6.搜索框中输入Chinese，然后选择【Chinese （Simplified）Language Pack for Visual Studio Code】，然后点击右侧的【Install】
7.安装好中文语言包之后软件会提示重启VSCode

```
